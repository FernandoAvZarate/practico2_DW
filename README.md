## Etapas del trabajo

### 1. Guía de estilos

Se elaboró una guía visual que incluye:

- Paleta de colores
- Tipografía principal y secundaria
- Logo e iconografía
- Componentes principales: barra de navegación, botones, formularios, tarjetas (cards), modal y footer
- Imágenes y carrusel (slider)

### 2. Prototipado

- **Wireframes de baja fidelidad** en Figma para todas las páginas (versión escritorio).
- **Mockups de alta fidelidad** con color, imágenes, texto real y componentes interactivos.
- **Prototipo navegable** en Figma (versión escritorio).
- **Mockup móvil** del index (versión responsive).

### 3. Codificación

El sitio fue implementado con:

- **HTML5** para la estructura.
- **CSS3 y Bootstrap 5** para el diseño y estilo.

El desarrollo refleja las decisiones tomadas en las etapas previas de diseño y prototipado.

## Responsividad

Todo el sitio es **completamente responsive**, asegurando una correcta visualización en:

- Dispositivos móviles
- Tablets
- Escritorios o notebooks
